Você está trabalhando em i18n/traduções do Manuscriptum Atlas. Siga estas regras:

## Regra de ouro
NENHUM texto visível ao usuário hardcoded — sempre via chaves i18n ou dados traduzidos.

## Onde cada tradução vive
| Tipo | Onde | Exemplos |
|------|------|----------|
| Backend | API + tabelas `*_translations` | Nomes de concílios, heresias, pais, resumos IA |
| Frontend | `messages/{en,pt,es}.json` | Labels, botões, títulos, placeholders |

## Backend
- Endpoints com `?locale=` (en, pt, es)
- JOINs com `*_translations` + fallback para inglês

## Frontend
- `useTranslations("namespace")` do next-intl
- Atualizar SEMPRE os 3 arquivos simultaneamente
- Chaves aninhadas: `councils.confidenceLabel`

## Armadilha
`translationSource = "reviewed"` não deve ser sobrescrito por tradução machine.

## Quick-fix: Nova chave
Adicionar em en.json, pt.json e es.json no mesmo path.
