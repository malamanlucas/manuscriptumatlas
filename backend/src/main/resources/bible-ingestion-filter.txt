# Bible Ingestion Filter
# =====================
# Controla quais livros/capítulos serão processados pelas fases de ingestão.
# 
# Formato:
#   BookName           → todos os capítulos do livro
#   BookName:3         → apenas capítulo 3
#   BookName:1-5       → capítulos 1 a 5
#   ALL                → todos os 66 livros (ignora o resto do arquivo)
#
# Linhas com # são ignoradas. Arquivo vazio = ALL.
#
# Exemplos:
#   John               → todos os 21 capítulos de João
#   John:1             → só João 1
#   John:1-3           → João 1, 2 e 3
#   Genesis:1          → Gênesis 1
#   Matthew            → todo Mateus
#
# Fases afetadas (APENAS Camada 4 — Alinhamento):
#   - bible_align_kjv, bible_align_arc69
#   - bible_align_hebrew_kjv, bible_align_hebrew_arc69
#
# Fases NÃO afetadas (processam TODOS os livros):
#   - Todas as fases das Camadas 1, 2 e 3
#   - Seed, texto, interlinear, léxico, glosses, enriquecimento
